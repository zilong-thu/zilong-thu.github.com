pull 操作

强制更新

Git Pull While Ignoring Local Changes?

【方法1】先clean当前的工作目录，使其与上一次的commit一致，然后再pull即可

【方法2】先将改动们暂存一下（stash），然后更新，更新完了再将暂存栈弹出或删除。
